# sos



We proudly secured 4th Rank at CGC (Chandigarh Group of Colleges) with our innovative Smart Women Safety Device.

This project is designed to enhance women’s safety through a compact, wearable emergency system connected to a dedicated mobile application. In critical situations, the user can press a single SOS button to instantly send alert messages along with live GPS location to pre-registered emergency contacts. The system also supports automatic SMS and call alerts for immediate response.

The device integrates ESP32, GSM, and GPS modules to ensure reliable communication and real-time tracking. The mobile application manages device connectivity, contact setup, and alert customization.

Our solution aims to provide a fast, simple, and effective safety mechanism that empowers women with confidence and immediate assistance during emergencies.
